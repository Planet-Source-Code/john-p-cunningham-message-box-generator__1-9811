Message Box Generator is a VB-6 Add-In
The file name is MsgBoxAddIn.dll
You must copy it to your Windows/Systems
Directory and then Register it, i.e.,
Click the Start Button, select Run and type in the
following:


  C:\Windows\System\REGSVR32.EXE MsgBoxAddIn.dll
